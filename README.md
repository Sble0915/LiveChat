# Java 기반 LiveChat 프로그램
해당 프로젝트는 대학 2학년 자바프로그래밍 수업의 텀프로젝트 과제로 인해 제작된 프로젝트 입니다.

## 프로젝트 목표 및 기능
- **통신 모듈의 추상화 :** 기본이 되는 통신 기능을 모듈화 하여 원하는 소켓으로 자유롭게 교체할 수 있는 구조
- **실시간 양방향 통신 :** 실시간 메시징 처리 및 동시성 제어가 보장된 브로드캐스팅
- **인프라 구축 :** 로컬이 아닌 실제 Ubuntu 서버(CLI 환경) 벡엔드 구동

## 기술 스택

### Backend
- **Language :** Java 25
- **Build Tool :** Gradle
- **Environment :** Ubuntu Server (CLI)

### Client
- **테스트 단계 :** HTML5, CSS3, JavaScript 등
- **최종 단계 :** Java25, JavaFX / Swing
